(2021) Portrait sketch by improving segmentation model





High-detailed Segmentation becomes Portrait sketch


The U-2Net is famous for segmetation model.

It's model has a complex structure so it does segmetnation to the level that presents a strand of hair.



Based on this high quality, people thought that it is possible to use this model for portrait sketch but at first it had not good performance.

I tried various trying to enhance this model such as preprocessing images, devising a dedicated augmentation techniques.

If you want to get information about U2Net, please refer this original repository

Enhancement of portrait sketchy

original repository proposed using of portrait sketchy with APDrawingGAN dataset, it is quite good for drawing portrait only but still it has bad performance, especially when it draws landscape.

So we enhanced it by using custom dataset and dataset transforms, such as random bluring, random affine, jittering head, jittering nearby human's outline.

But It still has many dirty messy pixels on the whole picture, we want to make it clear so applied Opencv based morphology, filling contour, blurring etc.

It became quite good, but we continued because we want to make it be lost more information such as face&cloth detail, but keep the important line information.
